\begin{abstract}

The report describes a limited application that performs ``live Twitter sentiment analysis''.
It can analyze the sentiment of tweets for a given hashtag, and plot the results on a website.
The performance of the system is quite limited, but it shows off some interesting properties and features of the Python language and its library ecosystem.
